PRIVACY POLICY

This Privacy Policy explicitly details what persistent information Nova Chat collects from your Discord guild ecosystem, how that specific data is maintained, and the clear rights you hold as a guild administrator or user.

1. Data We Collect & Scope

To deliver advanced community features and safety, Nova Chat processes essential data objects received from the Discord API structure. This is strictly limited to:

  • Discord Identifiers: User IDs, Server (Guild) IDs, Channel IDs, and Role IDs. These allow us to link permissions, configurations, and AI context accurately.
  • Message Metadata: Message IDs are temporarily checked solely to execute features like conversational AI responses and log configurations.
  • Configuration State: Custom settings and ticket system structures designed by server staff.

What We Do Not Collect: We never collect, cache, or ask for clear-text personal credentials, emails, authentication passwords, or financial transaction details.

2. How We Utilize Your Data

The operational data objects processed by the bot backend are reserved exclusively to fuel functionality components. We process information to:

  • Provide automated responses and verify server triggers.
  • Track engagement metrics to distribute activity rewards.
  • Troubleshoot technical faults, trace backend bugs, and mitigate widespread infrastructure abuse.

Nova Chat maintains a strict zero-monetization policy over user data. We do not sell, rent, license, or trade your server context or individual data to third-party companies.

3. Security Architecture & Storage

All server rules, configuration layouts, and user data are held securely using industry-standard database structures. Infrastructure networks are deployed via remote computing setups, applying standard firewall defenses to isolate sensitive cache elements.

4. Data Retention & Automatic Purging

We believe in maintaining data minimization practices. If Nova Chat leaves a Discord server guild voluntarily or via kick/ban actions, all localized server logs, configuration setups, and data indexes are automatically earmarked for deletion within 30 days.

5. Data Deletion & Privacy Rights

Server owners and individual users maintain full authority over their stored records. If you wish to manually initiate a wipe of your user footprint or stored server metrics, you can contact our technical center directly.

Please transmit your official request along with your respective Discord snowflake IDs to our email inbox: contact.novachat.bot@gmail.com. Requests are typically audited and processed within 7 business days.
